The Omnis

Every day, 14 workers die while on duty, with more suffering injuries. The Omnis is a combination of state of the art sensors and IoT software that can track a user’s vitals and location and transmit that data in real-time back to a central hub. The Omnis software then notifies a supervisor when a user is experiencing distress allowing for quicker response.
